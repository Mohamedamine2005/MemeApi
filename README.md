# MemeApi
Simple Meme Api Template (Free 2 Use &amp; Edit)

## Installation

Use the package manager [npm](http://npmjs.com/) to install packages.

```bash
npm i
```

## .ENV

```js
port=<SERVER PORT>
minutes=<MINUTES FOR RATE LIMIT>
maxReq=<MAX REQ PER MINUTES VALUE>
```
